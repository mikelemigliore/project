CSS Properties Made Easy

This website is made with the purpose to help the struggling beginners learning CSS syntax for the first time to memorize, but so have a guidance system and this site will help by furnishing a "dictionary" that will contain some of the most usefull properties that will help any programmer to style their own dream website

What will you find

As mentioned above, this websites contain the most used keywords in CSS. All the keywords are enclosed in the table below, with the words itself on the left and the corrisponding definition to the right, just like a dictionary. In the following website, you can practice all the concepts explained here and even more! Click Here!

Table of Properties

Keyword Definition
align-self Specifies the alignment for selected items inside a flexible container
background-color Specifies the background color of an element
color Sets the color of text
direction Specifies the text direction/writing direction
font-family Specifies the font family for text
@font-face A rule that allows websites to download and use fonts other than the "web-safe" fonts
padding A shorthand property for all the padding-* properties
position Specifies the type of positioning method used for an element (static, relative, absolute or fixed)
z-index Sets the stack order of a positioned element
scale Specifies the size of an element by scaling up or down
quotes Sets the type of quotation marks for embedded quotations
